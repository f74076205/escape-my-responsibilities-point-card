# 🐢 逃避集點手帳 v8.0 – 行為追蹤版

一個幫你 **溫柔紀錄「我在逃避什麼」與「我又逃去做了什麼」** 的小工具。  
不是要消滅拖延，而是陪你看見自己的模式，讓「休息」與「回歸」都被看見。

---

## ✨ 核心理念

- 接納「想逃避」是正常反應，而不是失敗。
- 把每一次想逃避，變成一個 **蓋章 + 行為紀錄** 的事件。
- 用可愛 UI + 圖表 + AI 小結語，幫你慢慢看懂：
    - 我最常逃避什麼？
    - 我通常為什麼想逃？
    - 我都逃去做了什麼？
    - 我有沒有順利回來？
        

---

## 🧁 主要功能總覽

### 1. 🧾 逃避集點卡（Stamp Card）

- 每張卡有 **12 個格子**，每次逃避會蓋一個章。
- 依照 **情緒強度 (1–5)** 自動選不同風格的印章：
    - 1：療癒小花小龜系 🌸🐢
    - 3：可愛點心安撫系 🍩🧸
    - 5：爆炸崩潰特效系 💢⚡🫠
- 卡片會自動計算目前是第幾張 `CARD #1 / 2 / 3...`。
    

---

### 2. ✍️ 當下紀錄：我在逃避什麼？

按下「💥 想要逃避一下！」會開啟輸入視窗：

1. **你在逃避什麼？**
    
    - 用 chip 按鈕選擇（例如：📚 看論文、💻 寫 Code、💬 回訊息、🧹 做家事…）
    - 可以點 `+` 新增自己的任務類型，會永久存在（存在 localStorage）。
        
2. **為什麼想逃？**
    
    - 用 chip 選擇原因（例如：😫 太累了、🥱 好無聊、😰 很焦慮、🧱 卡關了…）
    - 也可以用文字 `單次輸入其他原因…`。
        
3. **想逃的強度 (1–5)**
    
    - 用滑桿選擇，左邊是「還好」、右邊是「快爆炸」。
    - 強度會影響印章的圖案。
        

按下「蓋章並休息」後：

- 新增一筆逃避紀錄（taskType / reason / intensity / timestamp）。
- 螢幕會跳出 **3 分鐘小計時器**，陪你暫時抽離。
    

---

### 3. 🌳 3 分鐘情緒緩衝計時器

- 每次蓋章後會開啟「🌳 暫停一下」的 3 分鐘計時。
- 用來提醒自己：  
    →「我是在**有意識地**休息，而不是無限逃避。」
    
- 你可以：
    - 等時間到（計時結束後彈出提醒）
    - 或按「我準備好了（跳過）」提前關閉。
        

---

### 4. 👋 回來時的紀錄：我剛剛都在幹嘛？

系統會自動檢查最後一筆紀錄：

- 如果距離上次逃避 **超過 5 分鐘** 且還沒填「有沒有回去處理」，  
    會跳出 **「👋 歡迎回來」視窗**。
    

在這裡你可以：

1. **勾選剛才偷偷去做了什麼（可複選）：**
    
    - 例如：🪄 玩哈波、⚔️ 玩 FGO、📺 看哈波直播、▶️ 看 YT、💬 刷 Discord…
    - 同樣可以用 `+` 新增自己的分心行為。
    - 這些勾選會存進 `distractions` 陣列。
        
2. **回答：後來有回去處理那件事嗎？**
    
    - ✅ 有喔！
    - ⭕ 沒有耶  
        → 結果會記錄在 `returned` 欄位裡，用來計算回歸率。
        

---

### 5. 📊 右側數據儀表板

#### 5.1 統計小卡

- **修復回歸率**
    - 計算所有有填 `returned` 的紀錄裡，有幾次是 ✅ 回去處理。
    - 幫你看見「我其實回來了多少次」。
- **平均情緒強度**
    - 所有紀錄的平均強度（1–5）。
    - 越高代表壓力越重。
        

---

#### 5.2 圖表區（可點擊篩選）

1. 🥇 **任務 × 原因 長條圖（可點擊）**
    
    - 橫軸：任務類型（看論文、寫 Code…）
    - 堆疊顏色：不同逃避原因（太累、焦慮、卡關…）
    - 點一個長條 → 自動套用篩選，只顯示該任務＋原因的日誌。
        
2. 🎮 **逃避行為排行圖**
    
    - 統計 `distractions` 出現次數。
    - 看你最常逃去：玩遊戲、看直播、滑社群、看影片…
    - 這個圖幫你看見「我的主要分心出口」。
        
3. 🥈 **每日逃避趨勢（最近 7 天）**
    
    - 每日逃避次數折線圖。
    - 點某一天 → 只顯示那天的紀錄。
        
4. 🥉 **情緒強度分布圖**
    
    - 長條圖：強度 1–5 的出現次數。
    - 點強度長條 → 篩出所有該強度的紀錄。
        

> 被篩選時，上方會出現一個「篩選中…」標籤，可以隨時按 ✕ 清除。

---

### 6. 📅 逃避日誌（按日期摺疊）

- 以「日期」分組顯示紀錄：
    - 每組顯示：📅 2025/12/10（3 次）
- 每一筆紀錄包含：
    - 印章圖案（跟強度有關）
    - 任務標籤（📚 看論文…）
    - 原因標籤（😫 太累了…）
    - 分心行為標籤（例如：▶️ 看 YT、🪄 玩哈波）
    - 強度點點（●●●●●）
    - 時間（HH:MM）
    - 回歸狀態：
        - ✅ 有回去處理
        - ⭕ 沒有
        - ❓ 尚未回覆
    - 🗑️ 刪除單筆紀錄按鈕
        

---

### 7. 🤖 AI 模式洞察卡片

在側邊欄最上方有一張：**「🤖 AI 模式洞察」卡**

- 資料不足 5 筆時：
    - 顯示「AI 學習中，再累積幾筆我就會開始分析」。
- 資料足夠時，會自動分析：
    
    1. **你最常逃避的時段**
        
        - 例如：
            
            > 你的逃避好發期在下午 21:00–22:00。
            
    2. **你最常使用的分心行為**
        
        - 例如：
            
            > 你逃避時最常看 YT，要注意無意識的沉迷喔！
            
    3. **回歸率評語**
        
        - 回歸率高：稱讚你恢復能力很好
        - 回歸率低：鼓勵你，「回來就是勝利」
            

這張卡會隨著資料更新，給你「小小心理觀察」。

---

### 8. 📅 本週逃避週報（Report Modal）

按下「📊 生成週報」會打開一個：

> **📅 本週逃避週報**

內容包括：

- 本週逃避次數
- 平均情緒強度
- 本週最常見原因
- 本週最常見分心行為
    

下面還有一段 **AI 行動建議**，例如：

- 逃避頻率高 → 建議把大任務切碎。
- 常滑 Threads / YT / Discord → 建議設畫面時間限制。
- 常玩遊戲 → 建議玩完記得設鬧鐘，防止過頭。
    

---

### 9. ⚙️ 自訂選項與資料儲存

- 所有資料存在瀏覽器的 `localStorage`：
    - `escape_v8_data`：紀錄 stamps、客製任務、原因、分心行為。
- 不需要伺服器、無帳號系統，所有紀錄存在你的裝置裡。
- 右下有一顆「🗑️ 重置」按鈕：
    - 會清掉 `escape_v8_data`，等於重新開始。

## 🖥 使用方式

1. 下載或打開 `home.html`。
2. 用瀏覽器（Chrome / Edge 等）直接開啟。
3. 推薦將網頁 **加到桌面捷徑 / 書籤**，當成一個小小 App 使用。

## 🧩 技術簡介

- 純前端實作（不需要後端）
- 使用技術：
    - 原生 HTML + CSS + JavaScript
    - [Chart.js](https://www.chartjs.org/)：顯示統計圖表
    - `localStorage`：紀錄所有逃避資料與自訂選項
- 手風格：馬卡龍色系、手帳風、貼紙感 UI
